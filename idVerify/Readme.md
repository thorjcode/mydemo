# demo：身份证核验
### 大陆身份证核验算法实践实验
## 纯单机，无联网，无连接任何机关查询！！！！
### 导入 配置过程

1. 打开微信开发者工具，导入项目，填写APPID，编译运行；

2. 算法说明：将前17位数字分别乘以系数（7、9、10、5、8、4、2、1、6、3、7、9、10、5、8、4、2），将乘积结果相加后除以11取余数。余数0至10分别对应校验码1、0、X、9、8、7、6、5、4、3、2，其中余数为2时对应罗马数字X。例如前17位乘积和为189时，189除以11余2，对应校验码为X。校验码若计算结果为10则用X代替；

3. 此demo意为实践身份证算法，也可以验证他人提供的证件号码是否为符合国家标准的证件号码，但不能确认此号码为真实有效的证件！！！
